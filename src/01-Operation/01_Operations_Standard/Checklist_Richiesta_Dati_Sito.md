## 1. Antincendio (Obbligatorio)
- [ ] **Registro Antincendio**: aggiornato con le prove semestrali (UNI 11224).

- [ ] **Idranti/Naspi**: Verbale ultima prova di portata/pressione.

## 2. Idrico / Legionella (Obbligatorio)
- [ ] **Campionamenti**: Ultimo referto di analisi microbiologica (Ricerca Legionella).
- [ ] **Registro Autoclavi**: Verbale ultima manutenzione semestrale pompe/serbatoi.

## 3. HVAC / Climatizzazione
- [ ] **Libretto di Centrale**: aggiornato.
- [ ] **UTA/Filtri**: Verbale ultima sostituzione filtri (UNI EN 15780).

## 4. Elettrico / Elevatori
- [ ] **Quadri Elettrici**: Verbale ultima verifica periodica.
- [ ] **Ascensori**: Libretto e verbale ultima visita semestrale manutentore.
